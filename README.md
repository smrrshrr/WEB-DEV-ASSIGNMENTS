# WEB-DEV-ASSIGNMENTS

Gadget Launch Site – PlayStation 6

This project is a simple HTML-based webpage showcasing the rumored launch details, specifications, and pricing of the upcoming Sony PlayStation 6.

Features

Headings and structured content using semantic HTML.

Clickable image linking to a news source.

Unordered list for rumored specs.

Ordered list for performance highlights.

Table comparison of PlayStation 4, 5, and 6.

Price section with old price struck out and discounted price highlighted.
